Labels:    enhancement

Milestone: none

Body:
---
## Summary
The settings page currently only supports a light theme. Adding dark mode support would improve usability for users who prefer dark interfaces or work in low-light environments. This is a common accessibility and UX expectation for modern desktop applications.

## Proposed solution
Implement a dark mode toggle in the settings page that applies a dark color scheme to the UI. The setting should persist across sessions and integrate with any existing theme or appearance preferences in the application.

## Acceptance criteria
- [ ] Settings page includes a dark mode toggle option
- [ ] Selecting dark mode applies a dark color scheme to the settings page
- [ ] The dark mode preference is saved and restored on application restart
- [ ] Dark mode styling is consistent and legible (sufficient contrast)
- [ ] Light mode remains the default for existing users
